- `sp_60_13330_2020` → `ventilation_fosfokhim` (violations V04: кратность воздухообмена санузлов)

## Формат хранения

```
data/norms/
├── README.md              # этот файл
├── pue_7.pdf              # ПУЭ 7-е издание
└── sp_60_13330_2020.pdf   # СП 60.13330.2020
```

`norm_id` — snake_case, совпадает со значением в `ground_truth.expected_normatives` и полем `norm_doc` в violations.

## Требования

- Только официальные/общепринятые редакции
- PDF с текстовым слоем (не сканы)
- Русский язык
- Открытый/публичный доступ
